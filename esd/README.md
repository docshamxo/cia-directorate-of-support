# Executive Security Detail (ESD)

Close-protection unit under **OSEC** — HVT, senior leadership, and VIP security.

| Script | Channel purpose | Env var |
|--------|-----------------|---------|
| `coc.py` | ESD chain of command | `WEBHOOK_ESD_COC` |
| `information.py` | ESD mission / office information | `WEBHOOK_ESD_INFORMATION` |

```bash
python esd/coc.py
python esd/information.py
```
